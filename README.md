## Descrição
```bash
CRUD de usuários com e-mail, nome e senha.
```

## Instalação

```bash
$ npm install
```

## Executar o projeto

```bash
# dev
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
